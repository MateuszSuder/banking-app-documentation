# Proxy
*Funkcja, która wykonuje się przed każdym zapytaniem API. Podstawowy logger wyświetlający jaki path został zapytany.*
 
##### Mając - Zapytanie
##### Kiedy - Przed zapytaniem
##### Wtedy - Wypisz datę zapytania oraz endpoint zapytania.
